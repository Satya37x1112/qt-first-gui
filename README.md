# Qt First GUI

A beginner desktop GUI application built using C++ and Qt6.

## Features
- Interactive button
- Dynamic label update
- Custom dark cyber-style UI

## Tech Stack
- C++
- Qt6
- Qt Creator
- Linux
- CMake
